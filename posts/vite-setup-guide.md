# 使用 Vite 建立 React 專案
1. 打開終端機（Terminal），輸入以下指令：
```bash
# 使用 npm 建立專案
npm create vite@latest
```

2. 輸入專案名稱
![圖片](https://hackmd.io/_uploads/r1NbgHdDbl.png)

3. 選擇框架 (React)
![圖片](https://hackmd.io/_uploads/rJLVxBODWl.png)

4. 選擇語言 (Javascript / Typescript)
![圖片](https://hackmd.io/_uploads/BkRSlSuwbx.png)

5. 接下來根據需求選擇，通常初始練習可選 No。
 
6. 專案建立完成，下方有提示接下來要做的事：
![圖片](https://hackmd.io/_uploads/SyXseBOPbl.png)

7. 進入專案目錄：
```bash
cd [您剛建立的專案名稱]
```

8. 執行 `npm install` 安裝基礎套件：
```bash
npm install
```
![圖片](https://hackmd.io/_uploads/SJ64-ruwZe.png)

9. 執行 `npm run dev` 啟動開發伺服器：
```bash
npm run dev
```
![圖片](https://hackmd.io/_uploads/ryjt-S_vZg.png)

10. 進入提示的網址即可看到專案運行情況：
![圖片](https://hackmd.io/_uploads/HkBnbrOw-l.png)
